

  Type	    Meaning						                                   Example

* feat		A new feature						        	                feat(auth): add JWT token validation
* fix		A bug fix							                            fix(bookController): handle empty chapters array
* chore		Changes to build process, scripts, or maintenance		        chore: update dependencies
* docs		Documentation only						                        docs: add README section for API usage
* style		Formatting, whitespace, semicolons, etc. (no logic changes)	    style: fix indentation in server.js
* refactor	Code refactoring (no new features or fixes)			            refactor(bookController): simplify getBooks logic
* perf		Performance improvement						                    perf(db): optimize query for fetching books
* test		Adding or updating tests					                    test(bookController): add unit tests for getBooks
* ci		Changes to CI/CD config						                    ci: update GitHub Actions workflow
* build		Changes affecting build system					                build: update Webpack config
* revert	Reverting a previous commit					                    revert: feat(auth): add refresh token
* security	Fixing security issues					                        security: sanitize user input in createBook
